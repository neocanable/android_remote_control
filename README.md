# android 远程控制

**openstf的[minitouch], [minicap]二次开发**

功能:

- 将minicap和minitouch整合到一起。
- 通信完全走tcp，实现不插线，不用adb。
- 控制端提供websocket和tcp的接口，方便网页和pc客户端二次开发。
